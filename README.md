# Superstore Sales Analysis

## 📊 Project Overview

This project analyzes a retail sales dataset (Superstore) using Python. The goal is to identify key business insights such as sales trends, profit analysis, and regional performance.

## 🛠 Tools Used

* Python
* Pandas
* Matplotlib

## 📈 Analysis Performed

* Sales by Category
* Profit by Category
* Region-wise Sales Analysis
* Monthly Sales Trend
* Discount vs Profit Analysis

## 🔍 Key Insights

* Identified top-performing product categories
* Found regions with highest and lowest sales
* Analyzed impact of discounts on profit
* Highlighted loss-making products

## 📁 Dataset

* Superstore Sales Dataset (8000+ records)

## 🚀 Conclusion

This project demonstrates data analysis skills including data cleaning, visualization, and deriving business insights using Python.
